## tetris_rl
这是基于深度强化学习（DQN）来玩俄罗斯方块。

仓库内容说明：

- `src/tetris.py`：俄罗斯方块环境，实现了棋盘逻辑、方块旋转/碰撞、消行、渲染等（中文注释）。
- `src/deep_q_network.py`：简单的全连接 DQN（中文注释），输入 4 维特征，输出每个候选落点的 Q 值。
- `play.py`：加载训练好的模型并用贪心策略游玩（中文注释）。
- `train.py`：训练脚本，使用经验回放和 ε-贪心策略（中文注释）。
- `requirements.txt`：建议的依赖列表。

提供的best_model本地测试消除了上万行，效果不错。
